Calculator in C#

A simple C# calculator using the NuGet "xFunc.Maths" to calculate operations.

![Captura de tela 2024-08-22 150624](https://github.com/user-attachments/assets/8835ed4d-5fb3-42fa-b966-86ffcc625658)
